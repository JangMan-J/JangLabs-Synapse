## Already verified live this session (evidence in hand — phase b)

- `subscribe -p … --format json` — PUSH stream: `is_initial:true` full frame on
  connect, `is_initial:false` delta on change, tagged `pane_id`+`viewport[]`.
  Accepts MULTIPLE `-p` in one call. Replaces `dump-screen` polling on read path.
- `paste -p <chars>` — bracketed-paste text input, `-p`-targeted, isolation clean
  (marker 2× in target, 0× sibling).
- `send-keys -p <KEY>…` — KEY-NAME sender (`Ctrl c`, `Enter`, `Esc`, `F1`).
  NOT a text typer: multi-char token → `exit 2 unsupported key`. Complements
  `write-chars` for control/navigation keys (interrupt claude, drive codex gates).
- `run --block-until-exit[-success/-failure]` — synchronous exec primitive;
  blocked full 1.5s, exit 0 on completion.
